## Habit Tracker

### Features:
    ● User can add multiple habits
    ● Track each habit everyday and there are the 3 statuses of a habit:
        Done - Mark the habit as done for a day
        Not done - Mark the habit as not done for a day
        None - User did not take any action on a habit for a day
    ● A view to display 7 days of each habit
    ● Show today where user can mark todays habit and user can change today’s status as done, not done or none anytime.
    ● Show the previous 6 days and user can change the status of previous 6 days as well
